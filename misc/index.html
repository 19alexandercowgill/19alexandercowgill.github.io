This is Misc. It's not part of the main site

This section has tools that i find useful, and might be useful to you.

Decrypt

This is a tool in which you can input text and use a substitution cipher to mainly decrypt with a real time preview. It is useful, trust me.

Duck

This is a tool in which you can use DDGs API to get quick results

Markov

This is a tool in which you can input text and a rather primitive AI can do some somewhat simple text generation. It helps to have more text, but it doesn't matter as such, just decrease the context window.

Meta

This is a tool in which you can make Meta tags for your html pages.

UV

This is a tool in which you can make vector UV coordinates for your game in a Vector2f format (lwjgl)
