    Key Concepts:

    Codified - A Constitution in which all provisions are written down in a single set of documents, for example the US constitution.

    Uncodified - A Constitution that is not contained in a single set of documents, for example the British Constitution

    What is a Constitution?

    A constitution is a set of fundamental principles or established precedents that define the structure, powers, and duties of a government. It outlines the rights and responsibilities of citizens and the rules for how the state operates. Constitutions can be written (Codified) or unwritten (Uncodified), and they serve as the highest legal authority in a country.
